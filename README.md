# cbt (Cranberries Build Tool Version 0.2.0)

Build Tool for C/C++

# Usage

## cbt wandbox

'cbt wandbox' command allows to send your codes to wandbox and show program result.

Example

```cpp
// hello.cpp
#include <iostream>

int main(){
  std::cout << "Hello cbt!" << std::endl;
}
```

```
$ cbt wandbox cpp hello.cpp
hello cbt!
```

## Support Languages

- C

- C++

- Golang
